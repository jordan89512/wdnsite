Learning To Code

I started my journey into the coding world back in 2019. I wanted to find a way to express myself through programming because I think it's exciting to find logical solutions to everyday problems using these skills. I found software engineering to be an incredible creative outlet and an awesome in-demand skillset. I want to share the skills that I have learned and continue to learn as I walk the path of a Full Stack Software Engineer.

This Website is intended to be a free resource for students of Web Development. On this page you will find some of the learning resources and tools of the trade that I have used thus far. There are so many great online learning resources that are absolutely free and get straight to the point. So have a look around and I wish you the best of luck on your journey!

HTML

HTML or 'Hypertext Markup Language' is the skeleton of a website. It structures all of the images, elements and text on a webpage. It is the first thing you will need to learn.

CSS

CSS stands for 'Cascading Style Sheet'. It is used for styling your HTML documents and creating the final product of your webpage. CSS should be learned in conjunction with HTML.

JavaScript

Javascript is a scripting language used on the Web to enhance HTML pages. It is used to render web pages in an interactive and dynamic fashion.

UI/UX

User Interface and User Experience design concepts are crucial to understanding how to build quality layouts which conform to certain global standards on the web.

Responsive Design

In the early days of web design, pages were built to target a particular screen size. If the user had a larger or smaller screen than the designer expected, results ranged from unwanted scrollbars to overly long line lengths, and poor use of space.

As more diverse screen sizes became available, the concept of responsive web design (RWD) appeared, a set of practices that allows web pages to alter their layout and appearance to suit different screen widths, resolutions, etc.

It is an idea that changed the way we design for a multi-device web, and by looking through all of the free resources I have linked on this page, you will (with enough tenacity) understand the main techniques you need to know to master it.
